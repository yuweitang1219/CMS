開啟 Google Cloud Console

💡 Google API 設定步驟：

  1. 進入 Google Cloud Console 建立新專案。
  2. 啟用 Google Calendar API。
  3. 在「OAuth 同意畫面」設定為外部，並加入您的 Google 測試帳號。
  4. 建立「OAuth 2.0 用戶端 ID」，應用程式類型選擇「網頁應用程式」。
  5. 在已授權的重新導向 URI 中新增： http://localhost:8000/oauth2callback
  6. 複製 Client ID 與 Client Secret，填入上方並點選儲存。

💡 Line Bot 設定步驟：

  1. 登入 Line Developers 建立一個 Provider。
  2. 建立一個 Messaging API Channel。
  3. 在 Channel 設定中取得 Channel Secret。
  4. 在 Messaging API 頁籤最下方發行並取得 Channel Access Token。
  5. 將上方的 Webhook 網址複製，並貼到 Line Console 的 Webhook URL，並開啟 Use webhook 功能。
  6. 在 Line Developers 帳號設定頁面中找到您的 Your user ID 並填入上方（這能確保只有您能呼叫此機器人）。
  7. 儲存後，用手機掃描 Line Messaging API 的 QR Code 加入好友，即可透過對話傳送任務！
